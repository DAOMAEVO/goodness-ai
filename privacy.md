# Goodness AI（大善人AI）隐私说明

Goodness AI（大善人AI）是一款 Windows 本地工作台。本说明用于帮助用户理解使用软件时需要注意的隐私和安全事项。

## 本地使用

Goodness AI 主要面向本地使用场景。用户在软件中管理账号、API 密钥、平台配置和资源信息时，应自行确认数据来源、用途和保存方式。

## 敏感信息提醒

请妥善保管以下信息：

- 平台账号和密码
- API 密钥
- Token、Cookie 或访问凭据
- 个人联系方式
- 订单、渠道、货源或其他业务数据

请不要把包含敏感信息的截图、配置文件或导入数据公开发布到 GitHub、论坛、群聊或其他公开平台。

## 截图分享建议

如果你需要反馈问题或分享使用效果，请先处理截图中的敏感内容：

- 隐藏账号
- 隐藏 API 密钥
- 隐藏 Token / Cookie
- 隐藏个人联系方式
- 隐藏不希望公开的渠道和业务信息

## 安装包来源

请从本仓库的 Releases 页面下载安装包，避免使用来源不明的文件。

如果你从其他位置获得安装包，建议先核对版本、文件名和发布来源。

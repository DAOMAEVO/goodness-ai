# Goodness AI（大善人AI）

**Goodness AI，中文名：大善人AI**，是一款面向 AI 工具管理、账号批量处理、资源整理与善意分享的 Windows 本地工作台。

它希望把复杂的 AI 使用流程变得更清楚、更顺手，也希望让每一次有价值的整理、分享和互助，都能成为一份看得见、用得上的善举。

> 以善为始，以用为本。  
> 让工具服务人，让善意被看见，让有用的经验被传递。

## 产品定位

Goodness AI 不是单一脚本工具，而是一个本地化的 AI 工作台。

它适合用于：

- 管理 CAP / NewAPI 等账号与 API 密钥
- 批量导入、整理和维护平台账号
- 进行 AI 使用工具链转换与格式处理
- 管理 Codex / Cockpit 等本地 AI 工具配置
- 整理低价货源、渠道商家和本地提醒
- 共享可用资源，减少重复摸索
- 帮助新手更快完成基础环境和 GPT 使用准备

我们相信，真正有价值的工具不只是提高效率，也能帮助更多人少走弯路。把经验整理出来，把资源分享出去，把善意落到行动里，这就是 Goodness AI 想传递的方向。

## v0.1.6 更新亮点

本次版本继续围绕“本地化 AI 工作台”和“账号资源整理效率”补齐常用流程：

- 新增 Codex 配置中心，可在本地集中管理配置，并支持接管 `cc-switch`
- 增强本地化 Cockpit 管理，支持 `cpa-apikey` 一键导入与 Codex 接管流程
- 完善 NewAPI 账号管理，支持定时刷新额度与本地消息通知订阅
- 强化共享号池导入能力，支持批量导入、选择中转库和一键导入中转库
- 增强低价货源区，在线融合货源支持自定义接入、本地提醒和渠道商家整理

## 善意理念

Goodness AI 的中文名叫 **大善人AI**。

这个名字不是为了夸张，而是想表达一种朴素的愿望：工具可以更好用，人也可以更愿意互相帮助。

- 用 AI 降低复杂工具的使用门槛
- 用整理帮助他人节省时间
- 用分享减少重复踩坑
- 用清晰的工具承载更多善意行动

善不是一句口号。善可以是一次耐心整理，一次资源共享，一次对新手的帮助，也可以是把复杂流程做成一个普通人看得懂、用得上的工具。

**让善意更具体，让工具更温和，让有用的东西传到需要的人手里。**
